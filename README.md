# LingaTrans - Premium Chrome Translation Extension 🌍✨

LingaTrans is a professional-grade Google Chrome extension designed for seamless, high-performance text translation directly on any webpage. Built with **Manifest V3** and featuring a modern, glassmorphic UI, it provides a premium experience for both casual users and developers.

## 🚀 Key Features

- **✨ Magic Selection Button**: A smart, floating action button that appears instantly when you select text, providing one-click translation without cluttering your screen.
- **🛡️ Isolated UI (Shadow DOM)**: Uses advanced Shadow DOM technology to ensure the translator is never blocked or broken by a website's own styles.
- **📦 Instant Tooltip Results**: Translations appear in a sleek, blurred tooltip right next to your cursor.
- **🔄 Multi-Language Support**: Supports 20+ of the most common world languages (English, Arabic, Urdu, Chinese, French, etc.) with smart auto-detection.
- **📋 Copy to Clipboard**: A built-in copy button allows you to instantly copy translated text for use elsewhere.
- **💎 Premium Aesthetics**: Designed with modern design principles, including glassmorphism, smooth micro-animations, and a curated color palette.

## 🛠 Technology Stack

- **Manifest V3**: Compliant with the latest Chrome extension standards.
- **Shadow DOM**: Complete isolation from host webpage CSS.
- **Vanilla JS & CSS**: High performance with zero external dependencies.
- **Unofficial Google Translate API**: High-speed, free translation layer.

## 📥 Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/majorzzz/Translation-extenstion.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked**.
5. Select the `Translation-extenstion` folder.

## 📖 How to Use

1. **On-Page Selection**: Simply highlight any text on a website. Click the floating **"A" icon** that appears above your selection.
2. **Instant Translation**: The result will appear in a tooltip. You can change the target language or copy the text directly from here.
3. **Manual Popup**: Click the LingaTrans icon in your browser toolbar to open the manual translation box.
